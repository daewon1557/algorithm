## 진행 방식

- 매주 한 문제씩 조사를 해서 풀이과정과 함께 PR을 보낸다.

- 혼자 힘으로 풀어보고 안되면 풀이과정(README)을 참고한다.
- 풀이과정을 보고도 이해가 안되면 문제를 소개해준 사람에게 물어본다.
